• Lock a set of perks by clicking a lock bucket and selecting perks
    • If a perk doesn't appear it means that you own no armor with that perk
    • To use armor with multiple perks together shift+click the desired perks
    • - Armor with any of these perks will be used ("or")
    • - Armor with all of these perks will be used ("and")
  • Drag and drop items into the locked buckets to build sets with only that gear
  • Shift+clique em um item para construir sets sem aquele equipamento
  • Tiers mais altos são melhores
    • 4/5/2 (um set Tier 11) significa 4 Intelecto, 5 Disciplina, 2 Força (4+5+2 = Tier 11)
    • Intelecto acelera o tempo de recarga da Super.
    • Disciplina acelera o tempo de recarga de granadas.
    • Força acelera o tempo de recarga do corpo-a-corpo.
  • Tente encontrar uma armadura com perks que aumentem a quantidade de munição das armas que você mais usa.
  • Mude o Intelecto, Disciplina e Força dentro do jogo de acordo com o que é exibido ao criar cada Loadout.
  • Quando o nível de defesa de um item aumenta, os atributos deste item (intelecto, disciplina e força) também aumenta.
